Access Domains

A new concept to the IAM world, Access Domains are a way of grouping a number of hosts in which access should be consistent. This is a gap we've seen in larger organisations where existing IAM solutions focus on an individual application/host which can hinder processes like DR.

Access Roles

Portunus leverages the concept of Access Roles to bundle Access Items together. This ensures a better experiance for the business and better governance over logical access for our application owners and managers. This quick animation will demonstrate how you can request an Access Role.
