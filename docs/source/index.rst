Welcome to Portunus's documentation!
====================================

Portunus is a modern Identity and Access Management (IAM) solution designed to:

1. Scale to manage Internet Of Things (IOT)
2. Support unique and diverse infrastructure

Read through the documentation below for additional background in the Intro, as well as a practical user-guide.

Installation guides are provided for different scenarios (cloud vs on-premise)

The configuration section explains how to configure/set-up Portunus (e.g. adding custom agent interfaces)
